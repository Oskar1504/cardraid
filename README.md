# cardraid

## Installation
```bash
$ git clone https://github.com/Oskar1504/cardraid.git
$ npm install
$ npm run dev
```

## Usage/example

Small website using vue and bootstrap 5 hosted at http://localhost:3001/ .

If u just want to host the api delete public folder and change server.js
